Codificações de texto

Esses formatos são testados direto nos bytes e também por decodificação.

Compressão automática

Funciona para blocos/arquivos com assinatura ou formato reconhecível pelo navegador. Alguns formatos de jogos usam LZSS/LZ77/LZ4/custom sem cabeçalho; esses precisam de extrator específico.


Criptografia / ofuscação simples

Isto tenta métodos simples comuns. Criptografia real AES/Blowfish/TEA/etc. sem chave não tem como descriptografar automaticamente.


Tabela própria opcional

Para jogos com tabela própria, cole linhas no formato A=01, B=02, á=8A. A busca criará um padrão de bytes usando essa tabela.

Verdade sobre “todos os formatos”

Não existe ferramenta que leia todos os formatos, compressões e criptografias automaticamente. Jogos podem usar compressão própria, criptografia com chave, ponteiros, texto quebrado, imagens, scripts empacotados e tabelas customizadas. Esta versão tenta o máximo possível sem precisar instalar nada.
